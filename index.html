How to Use This Trainer Effectively

This trainer is designed for verbatim recall. To extract maximum value from the current feature set, use the following operating procedure:

1. Treat Each Card as a Full Paragraph Unit

The card title is your cue. Before revealing anything, attempt a full cold recitation with all text hidden. Verbatim accuracy — not recognition — is the target metric.

2. Use Word-by-Word Reveal as an Audit Layer

Click or press Space to reveal the text one word at a time. This surface is not for learning; it is for error detection. Identify where you substituted, reordered, or omitted words. These failures indicate weak segments that require targeted rehearsal.

3. Use Drill Mode for Stress-Proofing

The Drill tool flashes the text at you word by word. Use this once you have the sequence mostly stable. It forces your brain to recall content faster than your natural internal rhythm, strengthening the neural pathways. Use Space / → to advance, ← to go back, and ↓ to flag a struggle word (turns red).

4. Track Internal Progress, Not the Progress Bar

The progress bar reflects coverage, not mastery. The real milestone is: 100% correct recitation while the card is at 0% revealed.

5. Run Cards Forward in One Pass

Move sequentially through cards, overlapping cognitive load across paragraphs. This approximates a forward-run methodology.

6. Implement Reverse Runs Manually

Once forward recall is stable, manually run cards in reverse order (N → 1). This builds structural recall and eliminates flow-dependence.

7. Space Sessions According to Retention Best Practices

  • Initial session → Day 1 reactivation
  • Day 3–4 stability check
  • Day 7–10 consolidation
  • Day 30 retention audit

8. Rapid Reactivation Protocol (Manual/Drill)

For high-intensity verbatim memorization, use the Recall Scheduler. Each clean drill session (0 struggle words) advances the card through a graduated sequence: 1s, 5s, 10s, 30s, 1m, 2m, 5m, 10m, 20m, 30m. If you recall a card with struggle words, the schedule resets to ensure stability.

Wait for the UPS Yellow READY alert before re-testing a unit.

Spacing impacts consolidation more than session length.

9. Add Interference to Stress-Proof Recall

Enable Interference Mode in Drill view to add cognitive micro-tasks between each word reveal. Choose from Level 1 (gentle), Level 2 (moderate), or Chaos (random 1–15 words). This simulates the cognitive load of real-world driving recall scenarios.

Keyboard Shortcuts

All core interactions are keyboard-first:

  • Space / → — Reveal next word / advance in Drill
  • ← — Go back one word in Drill
  • ↓ — Toggle struggle flag on current word
  • A / B / C / D — Answer interference tasks
  • Escape — Close Drill mode
